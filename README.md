# Karta Rowerowa - Znaki Drogowe

Aplikacja do nauki znaków drogowych do karty rowerowej. Umożliwia przeglądanie znaków, naukę oraz rozwiązywanie quizów.

## Demo

https://delwing.github.io/karta-rowerowa/

## Technologie

- React
- TypeScript
- Tailwind CSS
- Vite

## Uruchomienie

```bash
npm install
npm run dev
```
